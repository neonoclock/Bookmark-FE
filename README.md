# 📚 Bookmark — 책 리뷰 공유 서비스 (Frontend)

**Bookmark** 는 읽은 책을 기록하고, 다른 독자들과 리뷰를 나누는 웹 서비스입니다.  
이 레포지토리는 **프론트엔드 전체**를 Vanilla JavaScript 기반으로 구현한 프로젝트로,  
Spring Boot 기반의 Backend API와 연동되어 작동합니다.

---

## ✨ 주요 기능 (Frontend)

### ✅ 1. 사용자 인증
- 회원가입 / 로그인 / 로그아웃
- 프로필 이미지 업로드(Base64 처리)
- 닉네임 및 프로필 정보 수정
- 비밀번호 변경 기능

### ✅ 2. 책 리뷰 기능
- 리뷰 목록 조회
- 리뷰 상세 조회
- 리뷰 작성 / 수정 / 삭제
- 이미지 포함 리뷰 작성 지원
- 본인 리뷰만 편집 가능

### ✅ 3. 커뮤니티 기능
- 다른 사용자의 리뷰 확인
- 좋아요 / 댓글 기능(필요 시)
- 취향 기반 추천 UI 확장 가능 구조

### ✅ 4. UI/UX 요소
- 다크 모드 대응
- 카드형 레이아웃 디자인
- 입력 검증 헬퍼 시스템
- 아바타 드롭다운 메뉴
- 부드러운 인터랙션 및 포커스 스타일

---

## 🛠 기술 스택

| 영역 | 사용 기술 |
|------|-----------|
| **Language** | HTML5, CSS3, Vanilla JavaScript (ES Modules) |
| **Build / 환경** | 단일 정적 웹서비스, Live Server |
| **State** | LocalStorage (세션 사용자 정보) |
| **API 통신** | Fetch API + JSON, Credentials 포함 |
| **구조화** | 모듈 단위 JS 설계(api / core / pages / common) |

---

## 🧩 핵심 구현 요약

### 🔹 1. 페이지 단위 모듈 구조
각 페이지는 `public/assets/js/pages/*.js` 로 분리되어 독립적으로 동작하며,  
공통 로직은 다음과 같이 모듈화하여 유지보수성과 재사용성을 높였습니다.

- **core/** → DOM / HTTP / Storage 핵심 유틸리티  
- **api/** → REST API 추상화  
- **common/** → 공통 UI(아바타 메뉴 등)  
- **pages/** → 페이지별 비즈니스 로직  

---

### 🔹 2. Fetch API 기반 통신
- `http.js` 의 `request()` 함수에서 모든 요청을 통합 처리  
- 실패 시 서버 메시지 정규화 후 오류 처리  
- `credentials: "include"` 로 세션 기반 인증 자동 처리  

---

### 🔹 3. 이미지 업로드(Base64)
- 프로필 이미지 및 리뷰 이미지 업로드를  
  **FileReader → Base64 → 서버 전송** 방식으로 구현  
- 이미지 미리보기 기능 포함  

---

### 🔹 4. Helper 기반 폼 검증 시스템
- `setHelper()`, `clearFormHelpers()` 로 input 오류 메시지를 통일적으로 관리  
- 모든 페이지에서 동일한 UX 품질 제공  

---

### 🔹 5. 아바타 UI 컴포넌트
- 로그인 상태에 따라 자동 UI 업데이트  
- 클릭 시 드롭다운 메뉴 오픈  
- 프로필 편집 / PW 변경 / 로그아웃 기능 포함  

---

## 🎨 UI/UX 특징

- 따뜻한 감성의 파스텔톤 중심 UI 디자인  
- 다크 모드 자동 대응  
- 카드형 리스트 기반의 확대/호버 인터랙션
